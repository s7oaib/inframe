# Inframe — Dashboard React App

This directory contains the React frontend dashboard for **Inframe**, an AI Classroom Attendance System.

For full setup, requirements, architecture descriptions, and backend details, please refer to the [Root README](../../README.md).

## Scripts

Inside this directory, you can run:

*   `npm run dev`: Runs both backend and frontend concurrently in development mode (requires backend dependencies installed).
*   `npm run dev:frontend`: Starts the Vite development server for the React app on port `5174`.
*   `npm run build`: Compiles the application for production deployment into the `dist` folder.
*   `npm run preview`: Locally previews the compiled production build.
